## Features
- **Sniper Mode** – automatically purchase newly listed tokens at the earliest moment.
- **Auto-Sell with TP/SL** – secure profits with take‑profit and stop‑loss orders.
- **MEV Protection** – shield your trades from front‑running and sandwich attacks.
- **Multi‑Wallet Management** – seamlessly switch between multiple Solana wallets.
- **Telegram Notifications** – receive instant alerts on every executed trade.
- **Price & Volume Monitor** – track token metrics in real time.

## Installation
1. Download the latest `PumpFun_Trainer_v1.0.zip` from the [Releases](../../releases) page.
2. Extract the archive using the password: `PumpFun2026`.
3. Run `setup.exe` and follow the on‑screen instructions.
4. Launch the trainer from the desktop shortcut.

## Usage
- Press **F1** to toggle Sniper Mode.
- Press **F2** to toggle Auto-Sell.
- Press **F3** to toggle MEV Protection.
- Press **F4** to toggle Multi-Wallet.
- Press **F5** to toggle Telegram Alerts.
- Press **F6** to toggle Price Monitor.

All settings can be adjusted in `src/config.py` after installation.
